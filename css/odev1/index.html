yerel sunucu kurma (programsız)

yerel sunucu kurmak için aynı modeme bağlanan birden fazla cihazınız olması gerekiyor örneğin bilgisayar cep telefonu tablet vs. benim durumumda bilgisayarı bir yerel sunucu yaptım nasıl mı !?

  1. Dosya ve yazıcı paylaşımını aç.
  2. internet information services 'ı aktifle.
  3. html dosyası veya web siteni paylaşma klasörüne ekle.
  4. bilgisayarın IP adresini öğren.

Dosya ve yazıcı paylaşımını açamk

  1. 'Denetim masası'nı aç
  2. 'Ağ ve internet'i seç
  3. 'Ağ ve paylaşım merkezi'
  4. 'Gelişmiş paylaşım ayarları'na girin
  5. Dosya ve yazıcı paylaşımı aç radyo düğmesine tıklayın

internet information services 'ı aktiflemek

  1. 'Denetim masası'nı aç
  2. 'programlar'i seç
  3. Windows özelliklerini aç veya kapat
  4. 'internet information services' kutucuğunu işaretle

html dosyası veya web siteni paylaşma klasörüne eklemek

  1. inetpub diye bir kkasör oluşmuştur onu aç
  2. wwwroot klasörünü aç
  3. web siteni buraya aktar benim durumumda odev1 adlı klasörümü aktardım sayfalar ve diğer dosyalar onun içinde , içndeki eski html sayfasını silebilirsiniz

bilgisayarın IP adresini öğrenmek

  1. Başlat menüsünün yanındaki arama kutusuna 'cmd' (komut istemi) yazıp 'Enter tuşuna basınız'
  2. 'ipconfig yazıp enter'a basınız' şöyle bir ekranla karşılaşırsınız
  3. cep telefonu gibi aynı ağa bağlanan bir cihazdan bilgisayarın IPv4 adresini tarayıcı çubuğuna giriniz , benim durumuda odev1 adlı bir klasörüm olduğu için

    ( IPv4 adresi) / odev1 /

    gireceğeim.
